# ALEXl0CO.github.io - Mi Portfolio

¡Bienvenido a mi proyecto de portfolio! Este repositorio contiene el código fuente de mi página web personal, donde comparto información sobre mí, mis habilidades y proyectos destacados.

## Descripción del Proyecto

Este proyecto tiene como objetivo presentar de manera visual y atractiva mi experiencia, habilidades y proyectos en los ámbitos de desarrollo web, diseño gráfico, etc. Utilizo tecnologías modernas para garantizar un diseño receptivo y una experiencia de usuario fluida.

## Tecnologías Utilizadas

- HTML5
- CSS3

## Estructura del Repositorio

- **index.html:** Archivo principal que contiene la estructura de la página.
- **styles/:** Carpeta que contiene los estilos CSS.
- **imgs/:** Carpeta que almacena imágenes y otros recursos multimedia.
- **README.md:** Este archivo que proporciona información sobre el proyecto.
